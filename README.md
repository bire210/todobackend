## This is a Todo app with the elastic Search feature


## Feature
  * Login
  * Sign
  * Authentication
  * Authorization
  * crud operation

    
## Run Locally
### Clone this Project
```
https://github.com/bire210/todobackend.git
```

### Install npm Packages

```
cd backend
npm i --global

```

### Run Server
```
npm start
```
